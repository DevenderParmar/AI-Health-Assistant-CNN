# 🩺 AI Health Assistant

An AI-powered web application that predicts the likelihood of diabetes using an Artificial Neural Network (ANN).

## 🚀 Live Demo

🌐 https://ai-health-assistant-ecru.vercel.app/

> Backend is hosted on Render's free tier, so the first request may take a few seconds.

## 🛠️ Tech Stack

- React + TypeScript + Vite
- FastAPI
- TensorFlow / Keras
- Scikit-learn
- Vercel & Render

## ✨ Features

- Diabetes prediction using ANN
- Real-time predictions
- Confidence score & risk level
- Responsive design
- FastAPI REST API
